# Hi there, I'm Petr 👋

**Senior Frontend Developer** · 🕑 UTC+2 (Central European Time)

I'm a **frontend engineer** with experience across several business domains — **B2B and B2C**: banking, investment platforms, loyalty programs, and EDMS (Electronic Document Management System). Most recently I've been working on **a factoring platform at a bank**, building complex web applications with React, TypeScript, and Next.js, and designing solutions based on **micro frontends** and **BFF (Node.js/NestJS)**.

My focus areas: performance optimization, frontend architecture, and shared module maintenance. I also do code reviews, mentoring, task breakdown, and engineering standards.
